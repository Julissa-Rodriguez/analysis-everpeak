# 📊 Análisis de Datos - ConnectaTel

## 🎯 Objetivo del Proyecto
Realizar un análisis completo de los datos de clientes de ConnectaTel para identificar patrones de uso, segmentar usuarios y proporcionar recomendaciones estratégicas para optimizar los planes de servicio.

## 📋 Datasets Utilizados
- **customers.csv**: Información demográfica de clientes
- **usage.csv**: Datos de uso de servicios (llamadas, mensajes, minutos)
- **communications.csv**: Registro de comunicaciones entre clientes

## 🔍 Etapas del Análisis
1. **Exploración inicial** de datos y estructura
2. **Limpieza de datos** (valores sentinel, fechas inválidas, nulos)
3. **Análisis exploratorio** con visualizaciones
4. **Segmentación de clientes** por edad y nivel de uso
5. **Identificación de outliers** y patrones extremos
6. **Análisis de correlaciones** entre variables
7. **Insights ejecutivos** y recomendaciones de negocio
